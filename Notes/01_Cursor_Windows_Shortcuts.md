# Cursor Keyboard Shortcuts (Windows)

Customize shortcuts: Ctrl + K, then Ctrl + S  
Or: Ctrl + Shift + P → type “Keyboard Shortcuts”

---

## Core AI / Agent

- Ctrl + I → Toggle Agent / AI side panel
- Ctrl + L → Toggle Chat (with selection, sends code to a new chat)
- Ctrl + K → Inline Edit (edit selected code with plain English)
- Ctrl + Shift + I → Open Agent in full-screen
- Ctrl + Shift + L → Add selected code to current chat
- Ctrl + Shift + K → Add selected code to Inline Edit
- Ctrl + . → Open mode menu (Agent / Ask / Plan / Debug)
- Shift + Tab → Cycle chat modes
- Ctrl + / → Switch AI model
- Ctrl + N → New chat
- Ctrl + T → New chat tab
- Ctrl + [ / ] → Previous / next chat
- Ctrl + E → Toggle Agent layout (side ↔ full)
- Ctrl + Enter → Accept all AI changes / run terminal command
- Ctrl + Backspace → Reject all AI changes
- Ctrl + Shift + Backspace → Cancel AI generation
- Alt + Enter → Ask in Inline Edit without applying
- Ctrl + Shift + Space → Voice Mode
- Ctrl + Shift + J → Cursor settings
- Ctrl + Shift + V → Paste as plain text in chat

---

## Tab / autocomplete

- Tab → Accept suggestion
- Escape → Dismiss suggestion
- Ctrl + Right Arrow → Accept next word only

---

## Chat helpers (type these in the chat box)

- @file / @folder → Add file or folder as context
- @Code → Reference a function or class
- @Docs → Use documentation
- @Git → Use Git changes / commits
- @Web → Pull info from the web
- @Cursor Rules → Include project rules
- @Past Chats → Use earlier chat history
- /summarize → Compress long chat context
- /commit → Generate commit message
- /pr → Generate PR description
- /create-rule → Create a Cursor rule
- /side or /btw → Side chat (without interrupting main agent)

---

## Everyday editor

- Ctrl + P → Quick open file
- Ctrl + Shift + P → Command Palette
- Ctrl + B → Toggle sidebar
- Ctrl + J → Toggle bottom panel
- Ctrl + ` → Toggle terminal
- Ctrl + \ → Split editor
- Ctrl + S → Save
- Ctrl + W → Close editor
- Ctrl + Tab → Switch tabs
- Ctrl + F → Find in file
- Ctrl + H → Replace in file
- Ctrl + Shift + F → Search in workspace
- Ctrl + D → Select next same word
- F12 → Go to definition
- F2 → Rename symbol
- Ctrl + / → Toggle comment

---

## Terminal AI

- Ctrl + K (in terminal) → Ask AI for a command
- Ctrl + Enter → Run the generated command
- Escape → Put command in terminal without running

---

## Tip

To check your own shortcuts: Ctrl + K, then Ctrl + S, search for `cursor`.
